# Testkey Signer

A magisk module or CLI tool to sign Android Verified Boot (AVB) images with test keys.

It was originally developed to apply OTA while keeping root and locked bootloader on Lenovo Tablets. But currently it is not user-friendly. I recommend to use [LTBox](https://github.com/miner7222/LTBox) instead.
